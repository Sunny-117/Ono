# AOP
  1. Aspect Oriented Programming
      层面     面向     编程

  2. 面向切面编程

  3. 在程序中时常用到的某些程序步骤、阶段、片段抽离出来，与本身的程序逻辑隔离

# 例子
  Node中读取文件、获取数据、处理数据、重写写入文件
  1. 读取文件获取数据
  2. 处理加工数据
  3. 重新写回文件
  读取与写入这个两个程序片段就可以被抽离出来

# 好处
  解决程序中的耦合问题，抽离和隔离出两个程序片段

console.log(1); // 固定逻辑
console.log(2); // 变化逻辑
console.log(3); // 固定逻辑
